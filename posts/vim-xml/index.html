Vim的xml排版

如果你手中有一个xml，而这个xml显示的排版非常混乱，大家看的时候可能会很为难，其实vim也可以实现xml的排版。

比如你的xml是这样：

需要安装一个程序。

如果使用的是fedora，输入 {% codeblock lang:bash %} yum install xml2 {% endcodeblock %} 如果是其他系统可能会叫xmllint

之后可以在终端中用命令排版， {% codeblock lang:bash %} xmllint -format -recover 后边加上文件名 {% endcodeblock %} 如果vim中使用，则需要修改~/.vimrc，添加如下： {% codeblock lang:vim %} map :silent %!xmllint -format {% endcodeblock %}

效果还不错吧～    以后就可以使用F2直接排版xml了～
